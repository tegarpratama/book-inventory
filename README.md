# Book Inventory App
Merupakan website sederhana yang dibangun dengan teknologi Node js-Express. Yang sebelumnya dibuat dengan [PHP Native](https://github.com/tegarpratama/inventory-toko-buku).

## Instalasi
Jalankan perintah :
```
npm install
```
Jalankan server: 
```
npm start
```
